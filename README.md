# fenix-games

Repositorio de juegos creado con Fenix MCP.

Fenix Games es una aplicacion Android nativa Kotlin, offline-first y de costo operativo mensual cero.

## Principios

- Sin Firebase, Supabase, Vercel, Render ni servicios cloud.
- Sin autenticacion remota ni telemetria externa.
- Persistencia local con Room, DataStore y JSON empaquetado/importable.
- UI con Jetpack Compose y Material 3.
- Arquitectura Clean Architecture con MVI.

## Ejecucion

El documento rector es `MANUAL_EJECUTABLE_FENIX_GAMES.md`.

Las validaciones se ejecutan por fase. Nunca se lanzan dos comandos pesados en paralelo.
